# 图片检测功能快速启动指南

## 功能概述

已将 YOLOv8 图片质量检测功能集成到民航检修记录系统中，用于自动识别检修照片质量。

## 架构说明

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   Vue 前端       │────│  Node.js 后端   │────│  Python 检测服务 │
│  图片检测页面    │    │  API 转发       │    │  YOLOv8 模型    │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

## 快速启动

### 1. 安装 Python 依赖

```bash
cd backend/python-services/image-detector
pip install -r requirements.txt
```

### 2. 一键启动所有服务

```bash
startall.bat
```

这将自动启动：
- 区块链节点 (端口 18545)
- 图片检测服务 (端口 5001)
- 后端 API (端口 3000)
- 前端界面 (端口 5173)

### 3. 访问图片检测功能

1. 登录系统：http://127.0.0.1:5173
2. 进入工作台
3. 点击左侧菜单"图片检测"

## 功能特性

- ✅ 批量上传图片（最多 20 张）
- ✅ 自动识别：normal（正常）/ bad（异常）
- ✅ 显示置信度百分比
- ✅ 网格展示检测结果
- ✅ 支持 jpg/jpeg/png 格式
- ✅ 文件大小限制 10MB

## API 接口

### 后端接口

```
POST /api/image-detection/detect
Content-Type: multipart/form-data
Authorization: Bearer <token>

file: <image_file>
```

```
POST /api/image-detection/detect/batch
Content-Type: multipart/form-data
Authorization: Bearer <token>

files: <image_files[]>
```

### Python 服务接口

```
GET  http://127.0.0.1:5001/health
POST http://127.0.0.1:5001/detect
```

```env
IMAGE_DETECTOR_URL=http://127.0.0.1:5001
```

## 权限要求

使用图片检测功能需要 `record.view` 权限，所有测试账号均可访问。

## 故障排查

### Python 服务无法启动

1. 检查 Python 版本：`python --version` (需要 3.8+)
2. 重新安装依赖：`pip install -r requirements.txt`
3. 检查端口占用：`netstat -ano | findstr 5001`

### 前端检测失败

1. 确认 Python 服务已启动
2. 检查后端日志
3. 验证文件格式和大小

### 模型加载失败

确认模型文件存在：`backend/python-services/image-detector/model/best.pt`
